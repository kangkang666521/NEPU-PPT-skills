# NEPU 模板选择

根据受众、场合和源内容选择合适的模板风格。

## 风格配置

| 配置 | 适用场景 | 视觉方向 |
|---|---|---|
| `petro-blue` | 学术汇报、课题组会、课程展示 | 简洁白底蓝调，清晰克制 |
| `nepu-red` | 校庆、学院大会、机关汇报、党建活动 | 庄重红色，工业仪式感 |
| `petro-gold` | 答辩、科研评审、重大项目汇报 | 深蓝底金色点缀，严谨正式 |
| `oilfield-green` | 学生活动、招生宣传、校园文化、社团展示 | 黑土绿调，青春活力 |
| `clean-white` | 文字密集的学术综述和研究梳理 | 极简留白，矩阵友好 |

## 内置模板映射

仓库 `assets/templates/nepu-civilization-office/` 下预置 7 套 PPTX 模板，与风格配置的对应关系：

| 内置模板 | 风格配置 | 场景建议 |
|---|---|---|
| 百廿红（李一） | `nepu-red` | 校庆、党建、学院大会、大庆精神/铁人精神主题 |
| 简单蓝（沈小丹） | `petro-blue` | 学术汇报、课题组会、课程展示（推荐默认） |
| 天空之境（潘冬远、张娉） | `petro-blue` | 科研汇报、学术报告 |
| 深海金芒（许歆瑶） | `petro-gold` | 答辩、重大项目汇报、科研评审 |
| 浩瀚星河（迮佳） | `oilfield-green` | 学生活动、校园文化、招生宣传 |
| 鎏金岁月（陈玥彤） | `petro-gold` | 答辩、结题汇报、颁奖仪式 |
| 诗意校园（徐臻） | `oilfield-green` | 社团展示、文艺活动、校园宣传 |

**使用规则**：
- 用户未指定模板时，根据场景从内置模板中匹配最合适的一套。
- `简单蓝（沈小丹）` 含 16:9 和 4:3 两种尺寸，默认用 16:9。
- 用户提供的授权模板优先于内置模板。
- 内置模板为便利性提供；商业/再分发使用前需确认权限。

## 决策规则

- 用户说"学术汇报""组会""课程展示"→ 优先 `petro-blue`，推荐「简单蓝」或「天空之境」。
- 用户说"答辩""评审""重大项目""结题"→ 优先 `petro-gold`，推荐「深海金芒」或「鎏金岁月」。
- 用户说"校庆""学院""党建""机关汇报"→ 优先 `nepu-red`，推荐「百廿红」。
- 用户说"学生工作""招生""活动""社团""校园文化"→ 优先 `oilfield-green`，推荐「浩瀚星河」或「诗意校园」。
- 源内容密集且用户未指定风格 → 默认 `petro-blue` + 简单蓝模板。

## 模板审计

当用户提供 PPTX 模板时，检查：

- 幻灯片尺寸和宽高比
- 封面/标题/内容/布局页
- 主题颜色
- 字体风格
- logo 位置
- 页脚/页码规范
- 典型图表/表格风格

然后复用该模板的语法，而非创造一套不相关的新风格。

## 无模板时的策略

1. 优先使用内置模板中匹配场景的那一套。
2. 其次让用户提供其院系/实验室的授权模板。
3. 如都不可用，以空白 16:9 PPTX 为基础，通过 NEPU 色板手建风格：
   - 封面：石油蓝背景 + NEPU logo + 标题。
   - 章节页：深蓝背景 + 白色章节标题。
   - 正文页：白底 + 石油蓝标题条 + 正文区。
   - 结尾页：与封面呼应。

## 大庆精神/铁人精神主题

- 主色调推荐 `nepu-red` + `petro-gold`。
- 优先使用「百廿红」模板。
- 可适当使用工业风格视觉元素（井架剪影线条、油田地平线）。
- 文字风格庄重有力，体现奋斗精神。
- 排版节奏可比学术汇报更富视觉张力。
